Introduction

Background

  • Data mining is finding valuable patterns in data.

  • It involves the use of computational techniques to extract high quality information from text.

  • Knowledge Discovery and Data Mining (KDD): “discovery by computer of new previously unknown information, by automatically extracting information from a usually large amount of different unstructured textual resources”

Features of Textual Data

  • High dimensionality.

  • Large number of features.

  • Multiple ways to represent the same concept

  • Highly redundant data

  • Unstructured data is easy for humans, hard for machine.

  • Abstract ideas hard to represent.

  • HOWEVER Huge amount of data to be processed –> Automation is required.

What Can TM Do?

1. Text Categorization

  • A new text is presented, and the objective is to attach the appropriate label to it.

  • Uses supervised learning techniques.

Source: Fundamentals of Predictive Text Mining

2. Information Retrieval

  • Information retrieval is the topic most commonly associated with online text.

  • Same idea as search engine.

  • You give the machine an input, it retrieves relevant text.

Source: Fundamentals of Predictive Text Mining

3. Clustering

  • Given a collection of texts, find a set of labels such that each holds similar texts.

  • Uses unsupervised learning techniques.

Source: Fundamentals of Predictive Text Mining

4. Information Extraction

  • Information extraction is a subfield of text mining that attempts to move text mining onto an equal footing with the structured world of data mining.

  • In terms of a spreadsheet model of data, the objective is to fill in the cells.

Source: Fundamentals of Predictive Text Mining

5. Evaluation and Prediction

  • Our ultimate goal is prediction, projecting from a sample of prior examples to new unseen examples.

  • The learning program studies documents and finds some generalized rules that will give correct answers on new examples.
